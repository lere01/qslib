# Code of conduct decision

- Status: accepted - external contributions closed
- Date: 2026-07-19

qslib does not currently accept unsolicited external contributions. The owner
confirmed this closed-contribution policy on 2026-07-19, so the project does not
adopt a public code of conduct or publish a conduct-reporting address at this
stage.

Issues, discussions, and pull requests do not grant contribution acceptance.
The maintainer may close unsolicited contribution proposals without review. A
personal email address must not be inferred from local Git configuration or
committed without consent.

Before opening the project to external contributions, a new governance decision
must require:

1. the owner approves a private reporting address or repository mechanism;
2. an explicit code of conduct is adopted with that contact;
3. `README.md`, `CONTRIBUTING.md`, and `SECURITY.md` link to it; and
4. the reporting path is tested without publishing confidential content.

Contributor Covenant 2.1 remains the preferred future baseline:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
