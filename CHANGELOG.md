# Changelog

All notable qslib changes will be documented in this file. The format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and stable releases
will follow [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Added

- Normative `qslib-conventions-v1` scientific and data conventions.
- Architecture, ADR, specification-first development, and TDD policies.
- Initial qslib 1.0 execution plan.
- Apache-2.0 licensing and pre-release project governance documents.

### Security

- External publication remains disabled. The approved identifiers are
  `qslib-quantum` for the Rust and Python distributions, `qslib` for the Rust
  library target, and `qslib_quantum` for the Python import package.
